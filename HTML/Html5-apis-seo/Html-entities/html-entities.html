HTML Entities

What Are HTML Entities?

HTML entities are used to display characters that have a special meaning in HTML, like < or >, or characters not easily typed on a keyboard.

Common HTML Entities

  • &amp; → &
  • &lt; → <
  • &gt; → >
  • &quot; → "
  • &copy; → ©
  • &reg; → ®

Example in Text

Displaying code: <p>This is a paragraph.</p>
